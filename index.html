Introdução

Este documento define as formas de acesso remoto aos serviços do TopSapp.
O TopSapp é um produto do Grupo MultiWare Tecnologia, com sede no município de Sinop/MT, que está pronto para te levar a vanguarda da tecnologia em Gestão de Telecomunicação, desde o ano de 2001, te levando com rapidez e segurança exatamente para onde você quer chegar.
O produto TOPSAPP, além de auxiliar na administração das empresas de telecomunicações, vem contribuindo com ferramentas que ajudam no melhor desempenho das redes de interatividade com os seus usuários finais.

Para utilizar essa API, será necessário ter o módulo contratado. Por gentileza, entre em contato com nosso comercial comercial@topsapp.com.br ou comercial1@topsapp.com.br

Esta API é REST e fornece os resultados em JSON, sendo adequada a todos os tipos de usuários, de baixo a alto volume, permitindo a consulta e utilização de diversas opções disponíveis no sistema.

URL Base


# Substitua "iplinuxluatpanel" pelo o endereço de acesso do seu servidor.
https://iplinuxluatpanel:9910/Login

O Endpoint base dependerá de cada servidor e será construído do seguinte modo:
https://iplinuxluatpanel:9910/


Login


# Exemplo em curl
curl -X POST -k -F 'usuario=xxx' -F 'senha=xxx' https://iplinuxluatpanel:9910/Login
-F 'identificador=xxx'

Realiza o login do usuário, retornando um código de sessão, que será desativada após 60 minutos de inatividade.
https://iplinuxluatpanel:9910/Login 



Resultado exemplo :
{
    "resultado":true,
    "codigo":"0",
    "msg":"Login realizado com sucesso", 
    "sessao":"f6b113b19eb70373010d0e5092467f3d", 
    "id_usuario":15,
    "nome_usuario":"TOPSAPP API"
}

PARÂMETROS

Campo Tipo Padrão Obrigatório Descrição
usuario String - Sim Operador de acesso do sistema TopSapp
senha String - Sim Senha de acesso
identificador String - Sim Identificador de origem do acesso que deverá estar liberado no TopSapp para conseguir consumir a API.

Erros

Possíveis erros e seus detalhes.

Código Erro Motivo
100 Não foi possível identificar sua sessão. Realize o login novamente. A sessão informada não é válida ou o tempo máximo de inatividade (60 minutos) foi atingido.
101 Erro ao realizar o login. O usuário e a senha não conferem com nenhum cadastrado no sistema.
102 O dispositivo deve ser liberado para acessar o aplicativo. O identificador utilizado não está liberado no sistema.
